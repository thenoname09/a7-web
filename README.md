KeenKeeper

📌 Description
KeenKeeper is a modern relationship management web app designed to help you stay connected with the people who matter most. It allows users to track interactions, monitor relationship health, and maintain meaningful connections through a simple and intuitive interface.

🚀 Technologies Used
Next.js (App Router)
React
Tailwind CSS
Context API (State Management)
React Icons

✨ Key Features
1. 📊 Relationship Dashboard

Track your connections with real-time stats like total friends, on-track relationships, and interactions.

2. ⏱️ Timeline Tracking

Log and view your interactions (Call, Text, Video) in a clean timeline format to never lose touch.

3. ⚡ Quick Check-In

Easily add interactions with one click using the Quick Check-In feature for faster updates.
